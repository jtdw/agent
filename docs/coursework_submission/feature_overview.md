# 功能清单与技术亮点

## 产品功能

- 数据上传：支持常见矢量、栅格和表格数据。
- 数据识别：识别字段、坐标系、范围和数据类型。
- 中文任务输入：用户可以用中文描述 GIS 分析需求。
- 智能工具选择：根据上下文选择 GIS 工具或稳定工作流。
- 地图预览：在前端展示空间结果。
- 结果面板：展示结构化摘要、任务状态和执行结果。
- Artifact 管理：生成的文件可以注册、展示和下载。
- 结果下载：支持地图、表格、矢量、栅格或报告产物下载。

## GIS 能力

- 矢量数据基本信息查看。
- 栅格数据基本信息查看。
- 表格经纬度转点。
- 矢量裁剪与制图。
- 栅格统计和栅格结果展示。
- 坐标系检查和重投影相关流程。
- 土壤湿度 XGBoost 建模。
- GeoConformal Prediction 不确定性输出。

## 工程亮点

- FastAPI 后端和 React/TypeScript 前端分离。
- Agent Runtime 将自然语言任务和 GIS 工具链连接起来。
- 工具执行前有参数、路径和文件存在性检查。
- 文件、会话、地图和 artifact 之间建立绑定关系。
- 支持中文和 UTF-8，降低 Windows 编码风险。
- CI 分层：轻量 PR gate 和手动/nightly full smoke 分离。
- Staging observation gate 可复现，包含 soil moisture/GCP gate。
- 保留 rollback 方案，不因本地通过就扩大真实流量。
