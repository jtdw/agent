# 拍摄分镜脚本

## 0. 准备

- 打开前端页面。
- 准备一份可演示的矢量、栅格或 CSV 数据。
- 准备三张图片：系统结构图、使用流程图、功能一览图。
- 视频建议录屏比例为 16:9。

## 1. 开场

画面：展示“GIS 智能体功能一览”图片或系统首页。

旁白：介绍这是一个 GIS 智能工作台，不只是聊天机器人。

时长：15 到 20 秒。

## 2. 展示整体界面

画面：录屏展示前端页面，依次指向上传区、对话区、地图区、结果区。

旁白：说明系统由文件上传、智能体对话、地图预览、结果面板和下载产物组成。

时长：40 秒。

## 3. 演示上传数据

画面：上传 Shapefile、GeoJSON、TIFF 或 CSV。

旁白：说明系统会识别数据类型、字段、坐标系和空间范围，并绑定到当前会话。

时长：40 秒。

## 4. 演示中文任务

画面：在对话框输入一个简单中文任务，例如：

```text
帮我查看这个数据的基本信息，并在地图上预览。
```

或：

```text
请把这个表格中的经纬度转换为点图层，并生成地图结果。
```

旁白：说明用户不需要直接选择底层函数，智能体会选择工具或工作流。

时长：60 到 90 秒。

## 5. 展示结果

画面：展示地图预览、结果卡片、artifact 或下载按钮。

旁白：说明结果不只是文本回答，还包括地图、文件产物、结构化摘要和可下载结果。

时长：60 秒。

## 6. 解释系统结构

画面：展示“GIS 智能体系统结构图”图片。

旁白：按前端、FastAPI、Agent Runtime、GIS 工具链、Artifact 管理、CI/Staging Gate 讲解。

时长：60 到 90 秒。

## 7. 解释使用流程

画面：展示“GIS 智能体使用流程图”图片。

旁白：从上传数据、识别数据、中文任务、工具选择、空间分析、地图展示、下载产物依次说明。

时长：60 秒。

## 8. 结尾

画面：回到系统首页或功能一览图。

旁白：总结项目完成状态：本地功能、CI、核心 gate 已完成，真实远端 staging 需要后续接入真实环境。

时长：20 到 30 秒。
