# GIS 智能体课程作业材料包

本目录用于课程作业视频讲解和项目提交。材料按“产品演示 + 技术结构说明”的方式组织，不包含答辩稿。

## 文件清单

- `video_script.md`：可直接照读或改写的视频讲解稿。
- `storyboard.md`：拍摄分镜，说明每一段展示什么界面、配什么旁白。
- `project_submission_intro.md`：课程作业提交用项目介绍。
- `usage_guide.md`：面向使用者的启动与操作说明。
- `feature_overview.md`：功能清单、技术亮点和交付状态。
- `visual_assets.md`：已生成图片的用途说明和本机路径。

## 建议视频长度

建议控制在 5 到 7 分钟：

1. 项目一句话介绍：约 20 秒。
2. 产品使用演示：约 3 分钟。
3. 系统结构和流程说明：约 1.5 分钟。
4. 功能亮点和交付状态：约 1 分钟。

## 讲解口径

这个项目不是普通聊天机器人，而是一个面向 GIS 数据处理的智能工作台。讲解时重点放在：

- 用户上传地理数据后，系统能识别数据并组织 GIS 工作流。
- 用户可以用中文描述任务，智能体选择合适工具执行。
- 结果以地图、结果面板、artifact 和下载产物的形式呈现。
- 项目包含测试、CI、staging gate 和回滚边界，体现工程稳定性。

真实远端 staging 尚未执行，因为缺少 staging host、admin token 获取方式、服务重启方式和部署配置核验路径。课程展示中可以说“本地和 CI 可复现通过，远端 staging 已有 checklist，但未接入真实环境”。
